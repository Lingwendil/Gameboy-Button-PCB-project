# Gameboy-Button-PCB-project

This project aims to be a repository for various PCBs intended to allow a user to construct a high quality game controller using the Nintendo Game Boy as a basis. The primary goal is to create a gamepad that used an original or reproduction Game Boy shella and buttons and one of these PCBs in order to have the most authentic control scheme for emulation or virtual consol/Switch online gamin functionality. Eventual plans also include functionality with the original Super Famicom/Super Nintendo console in order to play with the original Super Game Boy.
